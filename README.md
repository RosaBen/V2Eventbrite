# eventbriteV2
it's an event app where user can add new future event, attend any future event on app

steps to access app:
instal gems with:
```bash
bundle install
```
migrate database:
```bash
rails db:migrate
```
check status is up
```bash
rails db:migrate:status
```
launch the server with
```bash
bin/dev
```
access url : http://127.0.0.1:3000/
